# Приложение для автоматизации ведомостей выплат

Откройте `index.html` в браузере и приложите один или несколько Excel-файлов.

## Какие данные нужны

Приложение умеет читать общий файл или несколько отдельных файлов. Например:

- файл с фактами и планами;
- файл со справочником работников и иерархией;
- файл, где все эти данные уже объединены.

Лучше всего использовать такие колонки:

| Поле | Пример названия колонки |
| --- | --- |
| Период | Период, Месяц |
| Работник | Работник, ФИО работника, Оператор, Сотрудник |
| ID работника | ID работника, Код работника, Табельный |
| Регион | Регион, Область, Город |
| Канал продаж | Канал продаж, Канал |
| Главный специалист | Главный специалист, Глав спец |
| Региональный менеджер | Региональный менеджер, РМ |
| Супервайзер | Супервайзер |
| Руководитель СП | Руководитель СП, Руководитель |
| План | План, План продаж |
| Факт | Факт, Факт продаж, Продажи, Выручка |
| Факт оттока | Факт оттока, Отток факт |
| План оттока | План оттока, Отток план |
| Снижение бонусов | Снижение бонусов, Классификация ФРОДа, Фрод |
| KPI | KPI, KPI % |
| Факт ШПД | Факт ШПД, ШПД факт |
| План ШПД | План ШПД, ШПД план |

## Как считается

1. Сначала считается выполнение каждого работника: `Факт / План`.
2. Затем факты и планы работников суммируются на главных специалистов.
3. После этого те же факты и планы агрегируются на региональных менеджеров, супервайзеров и руководителей СП.
4. Выплата считается как `Факт * ставка выплаты`.
5. Ставки можно менять в левой панели, результат пересчитается автоматически.

## Результат

Кнопка `Скачать ведомость` выгружает Excel-файл с листами:

- Ведомость;
- Работники;
- Руководители;
- По регионам;
- По каналам;
- Ставки.

Лист `Ведомость` оформлен по шаблону из примера:

- блок `Руководитель ЦРК` с ФИО и итогами за месяц;
- блоки `Главный специалист` с ФИО, работниками и итогами;
- колонки `Факт продаж`, `План продаж`, `Факт оттока`, `План оттока`, `Снижение бонусов согласно классификации ФРОДа`, `KPI`, `Факт ШПД`, `План ШПД`;
- строки `Общая сумма`, `% выполнения`, `Модификатор`, `ИТОГ`;
- сиреневые шапки, красные подписи ФИО/месяца и поля согласования.
